# 🎵 Bollywood Music Recommendation System

A Machine Learning-based Music Recommendation System that recommends similar Bollywood songs based on lyrics, singer information, and song metadata.

## 🚀 Features

- Content-Based Recommendation System
- TF-IDF Feature Extraction
- Cosine Similarity Matching
- Interactive Streamlit Web Application
- Song Recommendations with Similarity Scores
- Beautiful User Interface
- Real-time Recommendations

## 📂 Dataset

Dataset Used:

Bollywood Songs Dataset (2017-2023)

Contains:

- Song Name
- Singer
- Release Year
- Lyrics
- Thumbnail URL

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MeghanaKanchiboina1616/Music-Recommender-System.git
```

Move into project directory:

```bash
cd music-recommendation-system
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Streamlit App:

```bash
streamlit run app.py
```
